## ❓ Frequently asked questions

Do I need internet access?
The application requires the internet to pull data from cloud-based services like Notion. If you use local Excel files, you can work offline.

Does it overwrite my existing PowerPoint files?
No. The application creates a new file every time you run a generation task. It does not touch your existing documents.

Can I change the slide design later?
Yes. Every generated file is a standard PowerPoint document. You can open it in PowerPoint to edit text, move objects, or adjust colors using standard controls.

Is my data secure?
The application processes your data locally on your computer. It does not upload your private Excel files to external servers.

What happens if the data is messy?
The software includes a data cleaning step. If it finds empty cells or invalid characters, it will pause and ask you to fix the entry.

## 🕒 Troubleshooting

If you encounter issues, check these steps:

* App fails to launch: Restart your computer and ensure you have administrative rights.
* PowerPoint does not open: Ensure that Microsoft Office is activated on your machine.
* Notion shows an error: Check your API key. Generate a new key from your Notion settings if the current one expires.
* Charts look wrong: Ensure your Excel headers are in the top row. The software reads the top row to name your slides.
